\end{itemize}

Make sure that you:
\begin{enumerate}
\item Include your names, github usernames and email addresses
\item In the introduction, define the task as you understand it
\item In the background, describe existing work, and any resources you will use, cite them appropriately
\item In the approach, describe what you tried (link to the code, describe in detail)
\item In the results, give the results of your approach (using tables, figures, example output)
\item In the discussion, describe your analysis of the results, any problems you had, and ideas for future work
\item In the conclusion, wrap up what you have done
\end{enumerate}
